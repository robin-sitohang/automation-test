# Web Automation test

## How to setup

```
git clone git@github.com:robin-sitohang/automation-test.git

cd sqa-test

npm install
```

## How to run test

### Run all test

```
npm run test
```

### Run single test

```
npm run test -- --filter cermati.js
```
